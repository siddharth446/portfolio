Featured Project

MoviesHub - Netflix Clone

Complete Streaming Platform Built in 10 Days
50,000+ Movies Offline Downloads 7 Platforms Realtime Sync

Full Cross-Platform Streaming Experience

A production-grade Netflix clone demonstrating advanced Flutter capabilities and full-stack development expertise. Built from scratch in under 10 days, featuring complete backend integration with Appwrite Cloud.

Key Features

  • Cross-Platform: Android, iOS, Web, Windows, macOS, Linux - one codebase
  • TMDB Integration: 50,000+ real movies with posters, trailers, ratings, and genres
  • Advanced Video Player: Resume playback, quality selection, subtitle support
  • Offline Downloads: Download videos + metadata using isolates for background processing
  • Realtime Sync: Continue Watching, Watchlist, and Likes sync across all devices
  • Authentication: Google, Email/Password, and Anonymous login with session management
  • Responsive UI: Adaptive layouts for phones, tablets, desktops, and web browsers
  • Premium UX: Hero animations, shimmer loading, pull-to-refresh, deep linking
